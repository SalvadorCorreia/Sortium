# Sortium

Advanced library sorting for the Steam client.

Sortium is a plugin built for the Millennium framework. It aims to expand Steam's native sorting capabilities by integrating external game data directly into your library view. If you have a massive library from Steam Family sharing or bundles, Sortium helps you organize it based on actual gameplay metrics rather than just alphabetical order or release dates.

## Current Status

Sortium is currently in early development.

The Minimum Viable Product (MVP) will focus on:
* Adding a new custom sorting filter to the Steam library dropdown.
* Fetching and caching completion data for your visible games from external sources (such as HowLongToBeat or Steam Hunters).
* Reordering the library interface based on median completion times.

## Requirements

* Steam Desktop Client
* [Millennium Framework](https://millennium.web.app/)
