# A list with all the importable Millennium elements.
This file works as a useful list to quickly check whether something exists without having to open all dropdowns on the documentation website. It can also serve as a decent copy-paste block for LLM usage, or as a reference file for Agentic workflows.

During the beginning phases of the `Sortium` development cycle, I faced a pretty major roadblock. I did not know how to make the most use of the Millennium framework. This situation only became worse when I decided to test out programming with an agent for the first time in this project. It did not go well. Because the Agent wasn't aware of the benefits of the framework, it completely ignored them and ended up rebuilding the wheel multiple times. Please avoid falling into the same pitfalls as me and make sure that if you work within this framework, you make an active effort to work with it and not against it. The people who created Millennium did an amazing job at making our lives easier; please don't ignore their excellent work.

### Documentation Links

**TypeScript API (`@steambrew/client` & `@steambrew/browser`)**
Construct URLs using this format to find the documentation for any element:
`https://docs.steambrew.app/plugins/ts/[environment]/src/[category]/[element_name]`

* **Environment:** `client` or `browser`
* **Category:** `functions`, `variables`, `enumerations`, `interfaces`, or `type-aliases`

**Example:** `https://docs.steambrew.app/plugins/ts/client/src/variables/Button`

**Lua API (`millennium`)**
The Lua API uses a different link structure. Append the function name after a `#` at the end of the base URL:
`https://docs.steambrew.app/plugins/lua/millennium#[function_name]`

**Example:** `https://docs.steambrew.app/plugins/lua/millennium#ready`
